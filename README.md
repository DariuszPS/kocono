# 🎨 Kocono - Kobo Highlights to Apple Notes

Preserve your Kobo Colour highlights when exporting to Apple Notes.

## ✅ Features

- Upload KoboReader.sqlite (drag & drop)
- Preview books and highlights
- Download AppleScript with colors preserved
- Dark mode support
- FAQ section
- Error handling & loading states

## 🚀 Quick Deploy

```bash
npm install
npm run build
vercel --prod
```
